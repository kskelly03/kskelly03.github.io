Project Goals

Using the pantheon dataset and the biopic dataset, I hope to be able to create an algorithm that can predict the success of a biopic on a given historical figure. This can be used to answer the research question:

In the era of biographical films being made in hollywood, we have seen reenactments of the lives of figures like Elvis Presley, Robert Oppenheimer, Elton John, Bob Ross, and Phil Knight have great successes at the Box Office.Given a list of historical figures ranked by popularity, can I predict the success of a biographical film made on that person?

Dataset Descriptions

The first dataset I will be working with is MIT's comprehensive list of famous faces, known as the Pantheon. Pantheon uses Wikipedia entries mined from Wikidata and the English, French, German, Italian, Spanish, Portuguese and Swedish editions of Wikipedia. The dataset comprises of over 2.29 million individuals and is cross-verified. The dataset provides birth dates, gender, citizenship, occupations, and other details. It combines these details with WikiData and their own variables in order to produce HPI, a popular index measuring worldwide popularity score of the given page. The pantheon database has been visited extensively by researchers of all kind.

Future Modeling

Given the datasets that I have incorporated, I have identified two different predicitive models that I can look into. First is a linear-regression model for box office predictions. I would take independent variables like HPI, birthyear, and occupation from the Pantheon and budget and release-year from the biopics dataset. The dependent variable would either be box office return or popularity score from the audience. I would incorporate this model using linear regression for the independent variables to predict the dependent variables. Then I would use R-squared or mean-squared area to test my model.

My second idea would be a k-nearest neighbors regression function to predict biopic success. This would involve using box office data to set a boolean data point based on ROI that returns True if the movie is above the set ROI and False if the movie falls below this ROI line. I would use similar independent variables, but this time my dependent variable would be a boolean value. I would implement KNN to predict biopic success based on historical figure attributes and other relevant features. Metrics such as accuracy, precision, recall, and F1-score can evaluate the model's performance. Then I could pick arbitrary Pantheon figures and return True if the movie is worth producing.
